# Produktanforderungen

- iOS App (SwiftUI)
- Backend (FastAPI, Docker)
- Web Dashboard (responsive)
- KI-Services (OpenAI API)

- Tageszusammenfassungen (KI)
- Tripplanung + KI-Reiseführer
